## Syntax Highlighting

- **Keywords** — Portal cyan `#5bc4f5`
- **Functions & methods** — Portal orange `#f5873c`
- **Strings** — Fluorescent White-Blue `#c8eef8`
- **Types & constructors** — Aerial Faith Plate Blue `#9de0f0`
- **Numbers & booleans** — Caution Yellow `#e8d87a`
- **Properties & attributes** — Propulsion Gel Purple `#c4a8f0`
- **Comments** — Concrete Dust, italic `#596266`

## Installation

### Via Zed Extension Manager (recommended)

1. Open Zed
2. Open the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
3. Run `zed: extensions`
4. Search for **Aperture Science** and install it

### Manual installation

1. Clone this repository into your Zed extensions directory:

```sh
git clone https://github.com/your-username/aperture-science-zed \
  ~/.config/zed/extensions/aperture-science
```

2. Reload Zed or restart the editor.

### Activate the theme

1. Open the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run `theme selector: toggle`
3. Search for **Aperture Science** and select it
